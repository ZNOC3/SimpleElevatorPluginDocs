# Downloading plugin

>[!Note]
>Currently supported unreal engine versions:
> - UE 5.0
> - UE 5.1
> - UE 5.2

To download plugin use Epic Games Launcher, in library tab search for simple elevator plugin and download it for selected Unreal Engine version.
